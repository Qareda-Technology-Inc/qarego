# Platform alert sounds (admin)

Configure from **Admin → Settings**:

| Sound | Default path | Used by |
|-------|----------------|---------|
| Kitchen alert | `/sounds/new-order.mp3` | Merchant kitchen (new food orders) |
| Rider alert | `/sounds/rider.mp3` | Rider app (pending ride/food offers) |

Upload mp3/wav in admin or place files in this folder. They are served at `/sounds/…` by the API.
